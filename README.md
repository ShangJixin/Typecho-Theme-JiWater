## 特性
- 基于HTML5UP-Future-Imperfect制作,使主题天然为CSS3、HTML5主题
- 主题为自适应屏幕主题，无论是使用PC、PAD、手机都可以愉快访问

## 安装
- 下载好压缩包后，请将解压后的文件夹重命名为'JiWater'后再启用主题
- 主题使用之前，请去后台设置外观填写必要的东西
- 外观设置1-5为必填，不填则影响全站图片功能和侧栏功能
- 外观设置多说二级域名为必填项，否则将影响全站的评论系统
